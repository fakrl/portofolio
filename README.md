## 💡 Getting Started

### Run Locally
1. Clone the repo:

```
   git clone https://github.com/fakrl/portofolio.git
````

2. Open the folder:

   ```
   cd portofolio
   ```
3. Open `index.html` in your browser or use **Live Server** in VS Code.

### Deploy

You can host this portfolio for free using:

* **Netlify** (recommended — used for this site)
* **Vercel**
* **GitHub Pages**
